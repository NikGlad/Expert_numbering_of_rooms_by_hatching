1.	Приложение по расстановки номеров делает нумерацию помещений. Номера помещений создаются в слое экспертиза номера помещений, шрифт Times new roman размер 300, цвет красный, толщина по слою, MText. Нумерация начинается с 1 далее 2 далее 3 и так далее пока не закончатся штриховки помещений, коридоров, вестибюлей, тамбуров. Номер помещения ставятся в центре каждой штриховки на чертеже. Номеруются все штриховки помещения (синяя штриховка), коридоры (зелёная штриховка), вестибюли (фиолетовая штриховка), тамбуры (голубая штриховка). Каждая штриховка имеет свой уникальный номер
Команда     START_NUMROOMS
2.	На некоторых коридорах (вестибюлях, фойе, холле) сложной, длинной формы возможно, что номер этого коридора уйдёт в другое помещение, пройти по таким помещениям и проверить, чтобы номера были на месте.

3.	Вторая программа нумерация помещений на определённом слое. Надо выбрать слой на котором расставлять номера. При необходимости можно предварительно самим расставить точки (кружочки) по всем помещениям в определённом слое затем пронумеровать на этом слое.

При расстановке номеров можно вручную в программе изменить расположение номеров 
•  (setq *fz_num_offset_x* 100.0) — сдвиг вправо на 100 
•  (setq *fz_num_offset_y* -50.0) — сдвиг вниз на 50
